# Notes Pro 📝

## Installation rapide sur iPhone

1. **Ouvrez ce lien dans Safari sur votre iPhone**
2. **Appuyez sur le bouton "Partager" (□↗)**  
3. **Sélectionnez "Sur l'écran d'accueil"**
4. **L'app sera installée comme une app native !**

## Fonctionnalités

✅ **Éditeur de texte riche** - Polices, tailles, couleurs, styles  
✅ **Images déplaçables** - Drag & drop avec redimensionnement  
✅ **Mode sombre persistent** - Se sauvegarde automatiquement  
✅ **Fonctionne hors ligne** - PWA complète  
✅ **Optimisé tactile** - Conçu pour iPhone  

## Utilisation

### 📝 Édition
- **Touchez l'écran** pour commencer à écrire
- **La barre d'outils apparaît** automatiquement au-dessus du clavier
- **Sélectionnez du texte** pour le formater

### 🎨 Formatage  
- **Police & Taille** : Menus déroulants
- **B/I/U** : Gras, Italique, Souligné
- **Couleur** : Sélecteur de couleur
- **🌙/☀️** : Basculer entre mode clair/sombre

### 📷 Images
- **Bouton 📷** : Ajouter depuis galerie
- **Toucher l'image** : Sélectionner  
- **Faire glisser** : Déplacer librement
- **Poignée bleue** : Redimensionner
- **Touche Suppr** : Effacer l'image sélectionnée

### 💾 Sauvegarde
- **Automatique** à chaque modification
- **Bouton 💾** pour forcer la sauvegarde
- **📝 Notes** pour voir toutes vos notes
