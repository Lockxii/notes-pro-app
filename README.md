# Notes Pro 📝

Une application de prise de notes moderne avec éditeur de texte riche et support d'images, optimisée pour iOS.

## 🚀 Fonctionnalités

- ✏️ **Éditeur de texte riche** avec formatage en temps réel
- 🎨 **Personnalisation complète** : polices, tailles, couleurs, styles
- 📷 **Gestion d'images** : upload, redimensionnement, positionnement libre
- 💾 **Sauvegarde automatique** dans le stockage local
- 📱 **Optimisé pour mobile** avec interface tactile
- 🌙 **Mode sombre** pour un confort visuel optimal
- 📝 **Gestion de multiples notes** avec aperçu
- 📴 **Fonctionne hors ligne** grâce à la technologie PWA

## 📱 Installation sur iPhone

1. Ouvrez Safari sur votre iPhone
2. Allez sur : **[https://votre-username.github.io/notes-pro](https://votre-username.github.io/notes-pro)**
3. Appuyez sur le bouton "Partager" (icône carrée avec flèche)
4. Sélectionnez "Sur l'écran d'accueil"
5. Confirmez l'installation

L'application apparaîtra comme une app native sur votre écran d'accueil !

## 🎯 Utilisation

### Barre d'outils
- **Police** : Sélecteur avec polices iOS natives
- **Taille** : Menu déroulant de 10px à 64px
- **Styles** : Gras (B), Italique (I), Souligné (U)
- **Couleur** : Sélecteur de couleur
- **Images** : Bouton 📷 pour ajouter des photos

### Raccourcis clavier
- `Cmd+B` : Gras
- `Cmd+I` : Italique
- `Cmd+U` : Souligné
- `Cmd+S` : Sauvegarder

### Gestion des images
- Cliquez sur une image pour la sélectionner
- Utilisez la poignée en bas à droite pour redimensionner
- Faites glisser pour repositionner
- Appuyez sur Delete pour supprimer

## 🛠️ Technologie

- **HTML5** + **CSS3** + **JavaScript**
- **Progressive Web App (PWA)**
- **Service Worker** pour le fonctionnement hors ligne
- **Local Storage** pour la persistance des données
- **Responsive Design** optimisé iOS
